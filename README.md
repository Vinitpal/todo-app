## Tech Stack

Client:
- React
- Tailwindcss

Server:
- NodeJS
- Express
- postgreSQL



## Run Locally

Clone the project

```bash
  git clone https://github.com/Vinitpal/todo-app
```

Go to the project directory

```bash
  cd todo-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
